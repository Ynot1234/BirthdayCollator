BirthdayCollator is a lightweight full-stack application built with .NET 9, C#, and a React
(Vite) frontend. It retrieves and normalizes significant (round number or aged) birthday data
from date-based sources, applies consistent cleanup rules, and presents results through a
modern UI. Users can optionally generate AI-powered summaries for each person by
providing their own OpenAI API key.

The project emphasizes clean architecture, async workflows, reusable components, and
predictable data pipelines.
